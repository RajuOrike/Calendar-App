# Calendar App - Frontend Labs Assignment

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Run dev server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```

4. Deploy with gh-pages:
   - Set `homepage` in package.json if desired.
   ```
   npm run deploy
   ```
